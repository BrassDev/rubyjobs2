== Welcome to rubyjobs2

This is a simple rails job board from Ruby Ireland, you can see the running website at http://www.rubyjobs.ie.

=== Dev Environment Setup

 $ git clone git@github.com:GavinJoyce/rubyjobs2.git
 $ cd rubyjobs2
 $ bundle
 $ rake db:create
 $ rake db:migrate
 $ rake db:seed
 $ rails server
 
=== Running Tests

 $ rspec
